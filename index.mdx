---

WimExplorer gives you a full graphical interface to open, browse, and modify Windows image files (`.wim`, `.esd`, `.swm`). Extract files, add new content, replace existing entries, and delete items — all powered by `wimlib-imagex` under the hood, with real-time logs and a built-in terminal for advanced use.

## Getting started

<Steps>
  <Step title="Download and run the launcher">
    Run `WimExplorer-v1.1.exe`. The launcher verifies your system automatically — it will prompt to install Node.js if needed.
  </Step>
  <Step title="Open a WIM file">
    Type or paste the path to a `.wim`, `.esd`, or `.swm` file in the path bar, then click **Abrir**.
  </Step>
  <Step title="Select an image">
    If the file contains multiple images (e.g., different Windows editions), a selector appears — choose the one you want to explore.
  </Step>
  <Step title="Browse and edit">
    Navigate the folder tree, select files, and use the toolbar to extract, add, replace, or delete content.
  </Step>
</Steps>

<Note>
  WimExplorer requires **7-Zip** to display file sizes and modification dates. If it's not found on startup, you'll be prompted to install it automatically via `winget` or specify its path manually.
</Note>

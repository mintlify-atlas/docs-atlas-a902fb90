---
title: "Keyboard Shortcuts"
description: "Quick reference for all keyboard shortcuts in WimExplorer"
---

WimExplorer supports keyboard shortcuts for common actions. All shortcuts work within the main browser window while the application is open.

## Navigation

| Shortcut | Action |
|---|---|
| `Double-click` | Enter a folder |
| `Enter` | Open the WIM file when the path input field is focused |

## Selection

| Shortcut | Action |
|---|---|
| `Ctrl+A` | Select all visible files in the current folder |

## File Operations

| Shortcut | Action |
|---|---|
| `Delete` | Delete selected items (shows confirmation before proceeding) |
| `Drag and drop` | Import files or folders into the current WIM location |

## UI Controls

| Shortcut | Action |
|---|---|
| `Esc` | Close open modals and context menus |
| `Right-click` | Open the context menu for the selected item |

<Tip>
  Right-clicking a file or folder gives you direct access to **Extract**, **Replace**, and **Delete** without needing to use the toolbar. This is the fastest way to act on a single item.
</Tip>
